Expense Tracker merupakan aplikasi manajemen keuangan pribadi berbasis web yang dikembangkan menggunakan Laravel dan MySQL. Aplikasi ini dirancang untuk membantu pengguna mencatat setiap pemasukan dan pengeluaran secara mudah dan efisien. Pengguna dapat menambahkan transaksi, mengelompokkan transaksi ke dalam kategori tertentu, memantau total saldo, serta melihat laporan dan riwayat transaksi. Dengan antarmuka yang sederhana dan informatif, aplikasi ini membantu pengguna memahami pola pengeluaran dan mengelola keuangan pribadi dengan lebih baik
